# 数组

数组是一种线性表数据结构。它用一组连续的内存空间，来存储一组具有相同类型的数据

## 线性表结构

数组、链表、队列、栈

## 非线性表

树、堆、图

## 连续的内存空间和相同类型的数据

```c
a[i]_address = base_address + i * data_type_size;
```

## 低效的“插入”和“删除”

链表适合插入、删除

数组支持随机访问，根据下标随机访问的时间复杂度为O(1)

插入元素或删除的时间复杂度O(n)

## 为什么从0开始编写

1.每次随机访问数组元素都多了一次减法运算，对于CPU来说，就是多一次减法指令。
2.最主要的原因是C语言设计者使用了0作为开始下标。
